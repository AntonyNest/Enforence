# Зразки ТЗ

Ця директорія містить зразки технічних завдань (DOCX) для RAG бази знань.

## Формат файлів

- `.docx` файли технічних завдань згідно КМУ Постанова №205
- Файли зберігаються через Git LFS

## Додавання зразків

1. Розмістіть DOCX файли у цій директорії
2. Запустіть інгестію: `poetry run python scripts/ingest_samples.py`
3. Перевірте кількість чанків у Qdrant
